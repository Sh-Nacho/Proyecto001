Para poder salir adelante y no estar tan asfiados con el tema de las camas uci. Primero deberiamos de organizar bien como estas van a estar desplegadas en todo el hospital, tener una especie de plano con cada una de las camas uci. Las familias no van a poder entrar a la zona donde se encuentre internado el paciente para evitar la concurrencia y solo los personales autorizados van a poder ingresar.


To be able to get ahead and not be so suffocated with the issue of ICU beds. First we should organize well how these are going to be deployed throughout the hospital, have a kind of plan with each of the ICU beds.

Por parte del Estado, este deberia de involucrarse más en este sentido y dedicarla más importancia a la crisis sanitaria, que en ves de estar discutiendo por problemas de intereses. Este ente deberia invertir en la salud para poder tener un mayor espacio y comprar nuevas camas y ventiladores correspondiente a la unidad de cuidados intensivos. La ciudadania debe de estar inmune ante el Covid y distintas variantes, por ende deben de estar vacunados todos para poder tener un contacto mas cercano, es decir ir al trabajo.


On the part of the State, it should become more involved in this regard and devote more importance to the health crisis, rather than arguing over problems of interest. This entity should invest in health in order to have more space and buy new beds and fans for the intensive care unit. Citizens must be immune to Covid and different variants, therefore they must all be vaccinated in order to have a closer contact, that is, go to work.


Para poder lograr entenderse, debemos de crear una red de hospitales que busque una organisacion entre ellos, para que los hospitales se dividan la carga entre ellos y que ninguno este sobrepasado. In order to be able to understand each other, we must create a network of hospitals that seeks an organization among themselves, so that the hospitals share the burden among them and that none is exceeded.
